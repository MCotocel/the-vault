  • A network is a group of interconnected computers
    • They can communicate and share resources, data, and storage
  • Shared resources can include:
    • Folders
    • Printers
    • Software
  • Networks allow users to:
    • Access files on a central server
    • Exchange data
    • Backup computers
    • Send data to printers
    • Access the internet
    • Communicate via email and video
    • Play multiplayer games
  • Advantages:
    • Sharing resources saves money
    • You can access files from any computer that is in the network
    • Data is simple to backup
    • You can communicate between computers
  • Disadvantages:
    • Network hardware is expensive
    • Managing a large network is complex
    • Virus attacks can infect all computers
  • Devices that can be connected to a network include:
    • Desktops
    • Laptops
    • Tablets
    • E-readers
    • Gaming consoles
    • Fridges
