  • There are many threats to networks
    • Malware:
      • Files are deleted or are encrypted
      • Computer crashes and slows down
      • Internet connections becomes slow
      • Keyboard input is logged
    • Phishing:
      • Gains access to a victim's account
        • Can withdraw money or purchase items
      • Gains access to information
      • Identity can be stolen
      • This is done by claiming to be from a known, reliable service
        • Usually, they include an action, such as clicking a link
    • Brute force attack:
      • Guessing password
      • Data theft
      • Access to systems
      • Brute force attacks often start with a dictionary attack, where they use a dictionary to guess passwords by simply trying and making small modifications to words
    • Denial of Service (DoS):
      • DoS attacks bring down servers by flooding them with requests to overload them
      • DDoS (Distributed Denial of Service) attacks use a large amount of machines (a botnet) to launch requests simultaneously
      • These attacks can be minimised by:
        • Using firewalls to inspect incoming traffic
        • Monitoring the network
        • Anti-malware software to avoid servers becoming part of a botnet
      • These attacks result in:
        • Loss of access for customers
        • Loss of revenue due to downtime
        • Lower productivity
        • Reputation damage
    • Data interception:
      • Logins compromised
      • Data stolen
    • SQL injection:
      • Contents of databases tricked into being outputted
      • Data can be modified
      • SQL attacks can be prevented by input validation
  • The main weak point is people:
    • Not installing updates
    • Not updating malware
    • Not locking doors
    • Not locking devices
    • Leaving information on desks
    • Writing down passwords
    • Sharing passwords
    • Losing laptops
    • Not encrypting data
  • Attacks can also come from inside of an organisation
  • Physical security must also be considered:
    • CCTV
    • Door locks
    • Disabling USB ports
  • Network security can be enhanced through a variety of ways:
    • Firewalls:
      • Firewalls examine the traffic between a user's computer and the internet
      • It checks the incoming and outgoing data to see if it meets certain criteria
      • If the data doesn't meet the criteria, the traffic is blocked, and the user is warned
      • All the traffic is logged
      • A list of IP addresses that might be harmful is kept and blocks access to the mentioned sites
      • Firewalls can be bypassed
      • Firewalls can't help with carelessness
      • Firewalls can also be disabled, which leaves the device open to harmful traffic
    • Proxy servers:
      • Proxy servers are servers that sit between the client and a web server
      • This allows internet traffic to be filtered
    • Network access control:
      • Different access levels for different kinds of users
      • Logins and passwords can be required for different levels
  • Why is network security important?
    • Information is financially valuable
      • People can be identified based on their behaviours
      • This information can also sway political policies
    • Networks must always be available
  • There are different forms of attack:
    • An insider attack is when someone inside of an organisation gives details about the organisation to others, or steals information
    • A passive attack is when a hacker sniffs data sent over a network. This attack is difficult to detect
      • Passive attacks involve listening, and are difficult to detect
      • This includes:
        • Packet sniffing, where the attacker intercepts data
        • Man in the middle attacks, where the connection to internet is intercepted and data travels through the man in the middle
        • DNS spoofing, where the DNS lookup requests are sent to a rogue server
          • The user is then sent to a fake version of the site
    • An active attack is when someone uses malware or another method to compromise a device's or network's security
      • Active attacks involve actively attacking a network
      • This includes:
        • Intercepting data with an active man-in-the-middle attack
          • Users often send important information unencrypted over the network
        • Releasing malware into a network
          • Malware is short for 'malicious software'
          • Malware is usually installed accidentally when the users think the software is harmless
          • There are different forms of malware:
            • Viruses are small pieces of code that are injected into other 'host' programs
              • They spread via files
            • Worms are malicious programs that can execute without user interaction
              • They are self-replicating, and spread to other devices by themselves
            • Trojan horses are programs that the user believes does something else
              • For example, a 'driver' could actually be a trojan horses
            • Ransomware encrypts the files on an infected computer and demands a ransom
              • Sometimes, the program keeps the files encrypted even after the money has been transferred to the creator of the program
              • Ransomware sometimes doesn't encrypt anything, and just scares the user into paying
            • Spyware gathers information from a user by logging keys pressed, turning on the camera and microphone, and recording the screen
              • This is done to gather information, potentially to steal passwords
            • Rootkits are programs that modify the operating system to avoid detection by antivirus programs
              • They are difficult to remove, since they embed themselves into the operating system
            • Backdoors are programs that open up a path for programs and people to gain easy access to a device
              • This can lead to remote access, where someone in a different location can access the device
        • Launching a DoS attack against a network
        • Cracking a password with a brute-force attack
    • Social engineering is when a person is tricked into giving away information
      • This can be done by pretending to be either tech support, and asking for a user's password, or by pretending to be a user that forget their password
  • Unpatched software can cause security threats, due to code that can be exploited
  • USB devices can expose the network to malware
  • Devices can be protected from malware in the following ways:
    • Keeping programs up to date
    • Installing an antivirus
    • Implementing user access levels so that the malware can't spread or so that the malware can't install in the first place
    • Using a firewall
    • Exercising caution
