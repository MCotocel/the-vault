  • Factors that affect network performance include:
    • Number of devices and users, since the bandwidth is shared between devices
    • 53.10 - Transmission media, since the media used can result in slower transfer speed
    • Interference, since data transmission can be slowed down by interference
    • Obstacles, such as walls and doors can slow down the data transmission
  • Network speeds help you:
    • Decide whether you need to upgrade your hardware
    • Estimate how long it will take to download a certain file
    • Determine if the performance lives up to the promises made by the ISP (internet service provider)
  • Bandwidth:
    • Bandwidth describe the amount of data that can be carried through a network connection, usually per second
    • More bandwidth means faster data transfer
    • A higher bandwidth is desired
  • Latency:
    • Latency describes the time in milliseconds between a bit leaving one device, and arriving at another.
    • A lower latency is desired
  • Calculating time to transmit a file:
    • Before calculating, it is important to know the difference between GB/s and Gb/s
      • GB/s means bytes per second
      • Gb/s means bits per second
    • It is also important to know the difference between KB, MB, GB, and TB
      • KB - 1,000 (thousand) bytes
      • MB - 1,000,000 (million) bytes
      • GB - 1,000,000,000 (billion) bytes
      • TB - 1,000,000,000,000 (trillion) bytes
    • The formula for calculating the time (in seconds) is $\frac{size}{speed} = time$
      • The size needs to be converted to bits
