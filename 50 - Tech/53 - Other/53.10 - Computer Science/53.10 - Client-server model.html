  • Server:
    • A server is a computer, usually a powerful one, that provides a network with a service
    • Services can include:
      • File servers, to hold data
      • Email servers, to manage emails
      • Web servers, to host a website
      • Printer servers, to provide access to printers
      • Authentication servers, to check if clients can access certain resources or not
  • Client:
    • A client is a weaker computer that uses services the server provides
  • There are several steps in this model
    • The client makes a connection to the server
    • The client sends a request to a server
    • The server sends a reply, or passes on the request if needed
    • Sometimes, authentication is needed
  • Advantages:
    • All data can be backed up centrally
    • Security is better since data is in one location
    • Updates can be managed centrally
    • Users can use any client to log in and access data
    • Accounts can be managed centrally
  • Disadvantages:
    • If the server is down, services are also down
    • Server hardware is more expensive
    • A cyberattack can focus only on the server
    • Servers can slow down when there is lots of load on the server
