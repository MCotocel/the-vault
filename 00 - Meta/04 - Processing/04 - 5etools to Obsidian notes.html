5ETools to Obsidian notes

Site

  • Shift-select creatures from the site
  • Right-click them and click add
  • Click printer view at the bottom of the stat block
  • Set printer columns to one
  • Click download as markdown

Command line

  • Make a bestiary directory
  • Move the file to the directory
  • Cd to the directory
  • Run the command csplit bestiary.md /^___$/ '{*}'
  • Run the command rm bestiary.md
  • Run the command for i in $(ls); do mv $i "$(cat $i | sed -n '2p' | cut -c5-).md"; done
  • Run the command find . -type f -iname "*.md" | while read i; do sed -i '1s/^/#dnd\n\n/' "$i"; done

Obsidian

  • Create a directory for the source under 60.60 - Bestiary
  • Move all the files to the source directory
  • Get a listing of all the files in the directory, one per line, then copy it into an index
  • Add [ and ](%60%20and%20%60) to the start and end of the file names, removing .md
